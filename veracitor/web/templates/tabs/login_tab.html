Welcome to Veracitor

Veracitor is a web intelligence service enabling you to evaluate the trust you can put in publishers on the Internet.

Using an interactive network of newspapers, blogs (the network can be made to contain all sorts of publishers) and other users of Veracitor, you can request a trust estimation of the entities contained in this explorable network. It is also expandable by you using Internet-crawling, and automatically updated.
